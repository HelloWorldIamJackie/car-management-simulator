<h3>Command Examples:</h3>

Get tickets
```
getTicket 110 1f 111 gf 112 2f 113 3f 114 gf ...
```

Activate tickets
```
activateTicket 110 111 112 ...
```

Drive to waiting zone
```
wait 110 1f 112 2f ...
```

Drive to loading zone
```
load 110 1f 112 2f ...
```

Finish loading and leave the plant
```
leave 110 1f 112 2f ...
```

Clear all data
```
clear
```
